# Axis Reference / 軸リファレンス

[日本語](#日本語) | [English](#english)

---

## 日本語

### 軸1：abstraction_direction

| 値 | 意味 |
|---|---|
| `concrete_to_abstract` | 具体的な事象から概念・原則へ向かう |
| `abstract_to_concrete` | 概念・アイデアから実装・具体へ向かう |
| `stays_concrete` | 終始具体的な話の中にとどまる |
| `stays_abstract` | 終始概念・方針レベルにとどまる |

### 軸2：problem_style

| 値 | 意味 |
|---|---|
| `pivot` | 別の方法・代替案を先に探す |
| `fix` | 原因を特定して修正する |
| `delegate` | 解決を相手に委ねる |
| `suspend` | 判断を保留する |

### 軸3：perspective_taking

| 値 | 意味 |
|---|---|
| `spontaneous` | 自発的に他者の立場が出てくる |
| `reactive` | 促されたときに出てくる |
| `absent` | ほとんど出てこない |

### 軸4：face_strategy

| 値 | 意味 |
|---|---|
| `high_mitigation` | 配慮表現が多い（score: 0.7〜1.0） |
| `moderate` | 断定と配慮が混在（score: 0.4〜0.7） |
| `low_mitigation` | 断定・直接的な表現が多い（score: 0.0〜0.4） |

### 軸5：concept_distance

| 値 | 意味 |
|---|---|
| `near` | 隣接する領域どうしを結びつける |
| `mid` | やや距離のある領域を結びつける |
| `far` | 通常接続されない領域を結びつける |

### 軸6：evaluation_framing

| 値 | 意味 |
|---|---|
| `gain_first` | 肯定・方向確認を先に出す |
| `loss_first` | 問題点・懸念を先に出す |
| `neutral` | 情報提供型。価値判断を前面に出さない |
| `mixed` | 文脈によって変わる |

### 軸7：need_for_cognition

| 値 | 意味 |
|---|---|
| `high` | 自発的に問いを立て、複雑な考察を展開する |
| `moderate` | 必要に応じて熟考する |
| `low` | 簡潔な答えと明確な指示を好む |

### 軸8：integrative_complexity

スコア 1〜7 の数値で出力されます。

| スコア | 意味 |
|---|---|
| 1 | 一次元的。単一の視点のみ |
| 3 | 複数の側面を認識するが統合しない |
| 5 | 複数の側面を認識し、相互関係を考える |
| 7 | 多次元的な枠組みを構築する |

### 軸9：epistemic_curiosity

| 値 | 意味 |
|---|---|
| `interest_type` | 知ること自体が目的（「面白そう」が動機） |
| `deprivation_type` | ギャップを埋めることが目的（「わからないと落ち着かない」が動機） |
| `mixed` | 両方の動機が混在 |

## English

### Axis 1: abstraction_direction

| Value | Meaning |
|---|---|
| `concrete_to_abstract` | Starts from specific events, moves toward concepts and principles |
| `abstract_to_concrete` | Starts from ideas, moves toward implementation |
| `stays_concrete` | Stays in concrete, specific territory throughout |
| `stays_abstract` | Stays at the concept and policy level throughout |

### Axis 2: problem_style

| Value | Meaning |
|---|---|
| `pivot` | Looks for alternatives first |
| `fix` | Identifies root cause and corrects it |
| `delegate` | Defers resolution to others |
| `suspend` | Holds judgment for later |

### Axis 3: perspective_taking

| Value | Meaning |
|---|---|
| `spontaneous` | Others' perspectives arise naturally and unprompted |
| `reactive` | Emerges when prompted or when relevant topics arise |
| `absent` | Rarely appears |

### Axis 4: face_strategy

| Value | Meaning |
|---|---|
| `high_mitigation` | Frequent softening expressions (score: 0.7–1.0) |
| `moderate` | Mix of direct and mitigating language (score: 0.4–0.7) |
| `low_mitigation` | Direct, assertive language (score: 0.0–0.4) |

### Axis 5: concept_distance

| Value | Meaning |
|---|---|
| `near` | Connects adjacent domains |
| `mid` | Connects moderately distant domains |
| `far` | Connects domains not normally associated |

### Axis 6: evaluation_framing

| Value | Meaning |
|---|---|
| `gain_first` | Leads with affirmation, then adds concerns |
| `loss_first` | Leads with problems or risks |
| `neutral` | Informational; does not foreground value judgments |
| `mixed` | Varies by context |

### Axis 7: need_for_cognition

| Value | Meaning |
|---|---|
| `high` | Spontaneously generates complex questions and analysis |
| `moderate` | Deliberates when necessary |
| `low` | Prefers clear, simple answers |

### Axis 8: integrative_complexity

Numeric score from 1 to 7.

| Score | Meaning |
|---|---|
| 1 | One-dimensional; single viewpoint only |
| 3 | Recognizes multiple dimensions but does not integrate them |
| 5 | Recognizes multiple dimensions and considers their relationships |
| 7 | Constructs multi-dimensional frameworks |

### Axis 9: epistemic_curiosity

| Value | Meaning |
|---|---|
| `interest_type` | Driven by interest — knowing itself is the goal |
| `deprivation_type` | Driven by discomfort with gaps — filling them is the goal |
| `mixed` | Both motivations present |
